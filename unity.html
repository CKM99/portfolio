Unity Game for the Tina Kelly Brand

Unity Project Description

Below is a unity game that I created for a natural hair company called the Tina Kelly Brand

It's similar to a art gallery but more interactive and knowledgeable. It talks about her products and how they benefit curly hair

I used C# to create this project. Take note of the automatic interactions I was able to create when approaching different objects like the television and the bathroom!
